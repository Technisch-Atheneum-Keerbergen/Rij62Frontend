# Rij62 Frontend

## Stack

- Svelte (UI framework)
- Eslint (JS linter)
- Tailwindcss (CSS tool)
- Bun (package manager and builder)

## Install

Once cloned, install [Bun](https://bun.com/docs/installation) and run

```sh
bun install
```

## Developing

To start a development server:

```sh
bun run dev --open
```

### Recommended VS code extensions:

- [Svelte for VS Code](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode)
- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- [Tailwind for VS code](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)
- [Tailwind Fold](https://marketplace.visualstudio.com/items?itemName=stivo.tailwind-fold)

## Building

To create a production version:

```sh
bun run build
```
